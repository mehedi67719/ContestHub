## ğŸš€ Project Overview
**ContestHub** is built with **React (frontend)**, **Node.js/Express (backend)**, and **MongoDB** for database.  
The platform supports **three roles**: Admin, Contest Creator, and Normal User.  

**Users can:**
- Browse and search contests by type  
- Participate by paying entry fees via Stripe  
- Submit contest tasks  
- View winners and leaderboards  

**Contest Creators can:**
- Add, edit, and delete contests (before approval)  
- See submissions  
- Declare winners  

**Admins can:**
- Approve/Reject contests  
- Manage user roles  
- Manage contests  

All private routes are secured with **JWT**, and the platform includes **role-based dashboards, responsive UI, and SweetAlert2 notifications**.
